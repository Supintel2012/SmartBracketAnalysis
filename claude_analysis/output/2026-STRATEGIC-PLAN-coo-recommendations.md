## Appendix B: Bake-Off Methodology Template

### 2026 MDP vs. LLM Bracket Bake-Off

**Methodology:**
1. All brackets generated before Selection Sunday
2. All brackets published publicly (transparency)
3. Scoring: ESPN standard (1-2-4-8-16-32)
4. Percentile calculated against ESPN Tournament Challenge

**Categories:**
| Category | Method | Notes |
|----------|--------|-------|
| SmartBracket (Pure MDP) | RRToolbox optimization | No user input |
| SmartBracket (Personalized) | RRToolbox + 3 questions | Standard adjustments |
| Claude Sonnet | Identical prompt | Latest model |
| ChatGPT-4 | Identical prompt | Latest model |
| Gemini | Identical prompt | Latest model |
| Hybrid (LLM â†’ MDP) | LLM picks as locked, MDP fills | Novel approach |
| ESPN Smart Bracket | ESPN's tool | Free baseline |
| Chalk | All higher seeds | Naive baseline |

**Prompt for LLMs:**
```
Generate a complete 2026 NCAA Men's Basketball Tournament bracket.
```
